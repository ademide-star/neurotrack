# neurotrack
An AI-powered behavioral analysis platform for neuroscience researchers.
Built for Morris Water Maze (MWM) experiments — tracks rodent movement,
generates heatmaps, trajectory plots, and locomotion statistics in real time.

## ✨ Features

- 🎥 Multi-video upload and batch processing
- 🗺️ Real-time trajectory visualization with animation
- 🔥 Spatial heatmap generation
- 📊 Speed graphs and distance comparison charts
- 🎯 Zone analysis — center, periphery, platform time
- ⬇️ Export trajectory and heatmap as PNG
- ⬇️ Export all session data as CSV
- 🖥️ Desktop app (.exe) and web version supported


## 🚀 Getting Started

Designed for neuroscience labs running spatial memory experiments.
Upload your MWM video, and NeuroTrack Pro automatically detects
the animal, tracks its path, and generates publication-ready plots.
